# OLX Bot

Sends notifications

## Setup

### 1. Create and activate a virtual environment

```bash
python -m venv .venv

# On Windows
.venv\Scripts\activate

# On Unix or MacOS
source .venv/bin/activate
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

## Usage

```bash
py ./olx_bot.py
```
